# Goodreads

## General Project Description
This is a project that is aimed at crawling and scraping through the goodreads website and storing all of the data in a MySQL database. Eventually this data will be used to develop a data analysis and visualization of something on goodreads, possible review analysis or user connections.

##Projected Object Outline
Users
	# of books read
	# of reviews published
	# of friends
	Average book rating
	Is an author?
	List of books read
	List of books currently reading
	List of books want to read

Books
	Title
	Author
	Date Published
	Rating Average
	Reviews
	Genre
	Summary

Reviews
	Reviewer Name
	Content
	Rating
	"Favorite"?
